# Customer Feedback Sentiment Analysis

**Intern ID:** CITS4760

## Project Overview

This project performs sentiment analysis on customer reviews using Natural Language Processing (NLP) and Machine Learning. Reviews are classified as either Positive or Negative based on customer ratings.

## Features

* Data Cleaning
* Text Preprocessing
* TF-IDF Vectorization
* Logistic Regression Model
* Sentiment Prediction
* Data Visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Joblib

## Dataset

Women's Clothing E-Commerce Reviews Dataset
